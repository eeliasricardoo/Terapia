## Usage

Import icons directly from `lucide-react`:

```tsx
import { User, Search } from "lucide-react";

function MyComponent() {
  return (
    <div>
      <User className="h-4 w-4" />
      <Search className="h-4 w-4 text-muted-foreground" />
    </div>
  );
}
```
