# US-GEN-02: Central de Ajuda e Suporte

**Como** usuário com dúvidas ou problemas,
**Quero** acessar a central de ajuda ou contatar o suporte,
**Para** resolver minhas questões.

## Critérios de Aceite
- [ ] **FAQ:** Lista de perguntas frequentes (Pagamento, Como agendar, Problemas de vídeo).
- [ ] **Chat de Suporte:** Widget (Intercom/Zendesk ou chat interno) para falar com atendente humano.
- [ ] **Ticket:** Formulário de contato se offline.

## Especificação Técnica
*   **Frontend:**
    *   **Página:** `/ajuda`
    *   **Integração:** Script de chat de terceiro (ex: Crisp, Intercom) ou formulário próprio enviando para e-mail do suporte.
